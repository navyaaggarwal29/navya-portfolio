## 💻 Tech Architecture
- **HTML5** & **CSS3**: Built exclusively with raw semantics, Grid layouts, and custom `@keyframes` for performance-focused animations.
- **Vanilla JavaScript**: Engineered with hardware interaction—tracking X/Y mouse coordinates for 3D parallax elements and utilizing `IntersectionObserver` for clean scroll reveals.
- **Design System UI**: Deep space technical dossier aesthetic with dark navy (`#05070a`) structures and custom glowing coral (`#ff6c4b`) interface accents.

## ✨ Core Features
- **Cinematic Hero**: Integrated typing sequence processing and an interactive live-telemetry HUD.
- **Storytelling Timeline**: A responsive vertical stepper layout handling experience progression (SheFi Scholar, Hackathons, AI & ML).
- **Premium 3D Interfaces**: Project cards and Extracurricular buttons respond dynamically to mouse coordinates with drop-shadow `.interactive-tilt` logic.
- **Lightweight**: Zero major heavy frameworks or CSS dependencies used, ensuring pure raw load speeds.

## 🛠️ Running Locally
Simply clone the repository and open the file to see the interface rendered in any modern browser without requiring a backend runtime.

1. Clone repo:
   ```bash
   git clone https://github.com/navyaaggarwal29/navya-portfolio.git
   ```
2. Navigate into the folder and open `index.html` either double-clicking it, or using Live Server inside VSCode.

## 📱 Responsiveness
The stylesheet heavily integrates mobile-first design philosophy using specific `@media` queries targeting `1024px` and `768px` breakpoints. Multi-pane terminal containers and geometrical grids conditionally collapse into graceful flex columns upon shrinking.
